Fittable is an Angular / Typescript component, designed to handle custom table structures, with spreadsheet capabilities. It comes equiped with a dynamic table model representation, an operation execution mechanism and corresponding UI functionalities, all of which are highly adaptable.

Fittable's main module was developd in Angular, representing strictly the view of the application, while the view model and all other modules were written in Typescript, without any dependency to a GUI framework, making it possible to be utilized on the server side as well if required.
